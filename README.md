# Hello shell!!
## min
make a file named `hello.sh`  
and give it a power: `chmod 755 hello.sh`  
Inside `hello.sh`  
```
#!/bin/sh
echo 'Hello shell!!'
```
Then run by `./hello.sh`  

It returns `Hello shell!!`  

## echo
### options
`echo -e`: enables escape sq  
sequence: [`\t`,`\n`,`\c`]  
last `\c` prevent line break
